# E-Learning-Course-Player
Responsive sidebar menu with click-to-expand and hover-to-preview behavior. Supports icon-only collapse mode with floating submenus. Built using HTML, CSS, and jQuery.

This project implements a responsive, collapsible sidebar menu commonly used in dashboards and LMS applications.

Features

Click-based submenu toggle in expanded state

Hover-based floating submenu in collapsed (icon-only) state

Smooth animations using jQuery

Clean UI without layout shift

Single-file implementation (HTML + CSS + JS)

Easily customizable and framework-agnostic

Perfect for admin panels, ERP systems, LMS platforms, and SaaS dashboards.
